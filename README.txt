# Vertical-Number-Puzzle
样例1：输入以下竖式：

　兵炮马卒
＋兵炮车卒
─────
车卒马兵卒

（1）打开NumberPuzzle.exe
（2）单击“手动出题”
（3）单击“加法竖式”
（4）在左下方文本框中直接输入如下3行（可直接复制粘贴）：
--------------------------------------------------------------
兵炮马卒
兵炮车卒
车卒马兵卒
--------------------------------------------------------------
	可以在右方预览框中实时查看你输入的竖式。
（5）单击“确定”
OK.

或者：直接在主界面单击“导入Demo”->“导入Demo1(加法)”->“确定”。



样例2：输入以下竖式：

　　ＡＢＣ
×　　ＤＣ
─────
　ＤＥＡＣ
　７ＥＤ
─────
　ＦＤＢＣ

（1）打开NumberPuzzle.exe
（2）单击“手动出题”
（3）单击“乘法竖式”
（4）在左下方文本框中直接输入如下5行（可直接复制粘贴）：
--------------------------------------------------------------
ABC
DC
DEAC
7ED
FDBC
--------------------------------------------------------------
	可以在右方预览框中实时查看你输入的竖式。
（5）单击“确定”
OK.

或者：直接在主界面单击“导入Demo”->“导入Demo2(乘法)”->“确定”。
